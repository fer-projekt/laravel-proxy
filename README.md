## Development

Clone the package from the github:

```bash
git clone https://github.com/fer-projekt/laravel-proxy.git

cd laravel-proxy
```

Starting & stopping docker

```bash
docker-compose up -d
```

```bash
docker-compose down
```

Install dependencies via composer and testing:

```bash
docker-compose exec app bash
```

```bash
composer update
```

```bash
phpunit
```
